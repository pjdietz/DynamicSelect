Dynamic Select Demo

Normal

Click the button to populate the select. This example is "unfiltered" and sends an array of objects with value and label properties.

// An array of objects in the standar format.
[        	
	{ label: "Homer", value: "1" },
	{ label: "Marge", value: "2" },
	{ label: "Bart", value: "3" },
	{ label: "Lisa", value: "4" },
	{ label: "Maggie", value: "5" },
	{ label: "An optgroup", value: [
		{ label: "Lenny", value: "6" },
		{ label: "Karl", value: "7" }
	]}
]

Filtered

The DynamicSelect for this example is created with a filter function. The filter expects an object which it converts to an array of objects value and label properties.

// The object starts off like this...
{
   fry: "Philip J. Fry",
   leela: "Turanga Leela",
   // ...
}

// ...and the filter function turns it into this...
[
    {
        value: "fry",
        label: "Philip J. Fry"
    },
    {
        value: "leela",
        label: "Turanga Leela",
    },
    // ...
]
